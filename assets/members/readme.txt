WANT TO SHUFFLE THROUGH MULTIPLE PHOTOS PER MEMBER?
Make a folder with that same name (no extension) and number the photos
inside it 1, 2, 3, ... — for example:

  1_bang_chan/1.png
  1_bang_chan/2.png
  1_bang_chan/3.png

If a member has 2 or more photos this way, the app shuffles between them
every few seconds, never showing the same one twice in a row. A member with
only one photo (in a folder or as the flat file above) just shows that one.
The folder is checked first — if it exists, the flat single-file version
above is ignored for that member.
